# AI-Based Business Decision Support System

## 📌 Project Overview

The AI-Based Business Decision Support System is a Business Intelligence project developed using Amazon Sales data. The project demonstrates how data can be transformed into meaningful business insights through data preprocessing, visualization, and automation.

The dataset was cleaned using Microsoft Excel and analyzed in Microsoft Power BI to create an interactive dashboard. ChatGPT was used to generate business insights and recommendations, while Zapier was integrated to automate the flow of data between Google Forms and Google Sheets. Finally, all project files were managed and shared through GitHub.

---

## 🎯 Objectives

- Clean and preprocess Amazon sales data.
- Build an interactive Power BI dashboard.
- Analyze sales performance using KPIs.
- Generate AI-based business insights.
- Demonstrate workflow automation using Zapier.

## 💡 Business Insights

- Total Sales reached **₹10.72 Million**.
- Sports & Outdoors is the highest-performing product category.
- The United States generated the highest sales.
- Average Order Value is approximately **₹914.93**.
- Most customer orders were delivered successfully.
- Shipping costs are well controlled compared to total revenue.

---

## ⚡ Zapier Automation

Workflow:

Google Forms → Zapier → Google Sheets

Whenever a new Google Form response is submitted, Zapier automatically creates a new row in Google Sheets, reducing manual work and improving efficiency.
